# pitchpx-example-pitchfx
pitchpx example for pitch f/x

## Requirement

- python 3.4+(don't know about version < 3.4, sorry)

## Install

```bash
$ pip install -r requirements.txt
$ jupyter nbextension enable --py --sys-prefix widgetsnbextension
```

## Usage

```bash
$ jupyter notebook
```
